# 线性系统的能控性和能观测性

# 能控性

能控性的简单理解为输入能够控制系统的状态变量 x(t)x(t)x(t) 和输出变量 y(t)y(t)y(t)）。

例子

以下图结构图系统为例，输入 uuu 只能影响 x1x_1x1​ 而不能影响 x2x_2x2​ ，因此可以说该系统中 x2x_2x2​ 不可控， x1x_1x1​ 可控，系统整体不可控。

# 线性定常系统的能控性定义

对于线性定常系统：

x˙=Ax+Bu(1)\dot{x}=Ax+Bu \tag{1}x˙=Ax+Bu(1)

如果存在输入控制 u(t)u(t)u(t) 在有限的时间 [t0,t1][t_0,t_1][t0​,t1​] 内能将系统从初始状态 x(t0)x(t_0)x(t0​) 转移到任意的状态 x(t1)x(t_1)x(t1​) 。

# 线性定常系统的能控性判据

  1. 判据 1（PHB 判据）：对于式（1），系统，该系统完全能控的充要条件为能控性矩阵

Uc=[BAB⋯An−1B](2)U_c=\begin{bmatrix}B&AB&\cdots&A^{n-1}B\end{bmatrix} \tag{2}Uc​=[B​AB​⋯​An−1B​](2)

的秩为 nnn ，即 rankUc=nrankU_c =nrankUc​=n。

  • 对于线性系统，经过线性非奇异变换后，状态能控性不变。（即能控性与系统有关，与状态变量的选取无关）

# 线性定常系统的输出能控性

很多情况下，被控制量往往是系统的输出而非状态变量，因此还需分析输出能控性。

  1. 输出能控性的定义：对于线性定常系统： x˙=Ax+Bu\dot{x}=Ax+Bux˙=Ax+Bu， 如果存在输入控制 u(t)u(t)u(t) 在有限的时间 [t0,t1][t_0,t_1][t0​,t1​] 内能将系统从初始状态 y(t0)y(t_0)y(t0​) 转移到任意的状态 y(t1)y(t_1)y(t1​) ，那么可以说明系统是输出的完全可控。
  2. 输出可控性的判据：系统输出完全可控的充要条件是矩阵 [CBCABCA2B⋯CAn−1B]\begin{bmatrix}CB&CAB&CA^2B&\cdots&CA^{n-1}B\end{bmatrix}[CB​CAB​CA2B​⋯​CAn−1B​] 的秩等于输出的维数，即矩阵 CCC 的维数。
例题

# 能观测性

能观测性是指通过对输出的测量来确定系统的状态变量，反映了从系统外部观测系统内部的能力。

在前面状态方程的解方面的内容中，我们知道线性时不变系统动态方程（取初态时间 t0=0t_0=0t0​=0）的解为：

x(t)=eA(t)x(0)+∫0teA(t−τ)Bu(τ)dτ,t≥0(3)x(t)=e^{A(t)}x(0)+\int_{0}^te^{A(t-\tau)}Bu(\tau)\,d\tau, t\geq0 \tag{3}x(t)=eA(t)x(0)+∫0t​eA(t−τ)Bu(τ)dτ,t≥0(3)

上式中，系统矩阵 A,BA,BA,B 以及控制输入 u(t)u(t)u(t) 已知，因此只需要关注初态 x(0)x(0)x(0) 即可得到 x(t)x(t)x(t)。

# 线性定常系统的能观性定义

对于线性定常系统：

{x˙=Ax+Buy=Cx(4)\left\{ \begin{aligned}\dot{x}&=Ax+Bu\\y&=Cx \end{aligned}\right. \tag{4}{x˙y​=Ax+Bu=Cx​(4)

如果在任意给定的输入 u(t)u(t)u(t) 下，根据输出 y(t)y(t)y(t) 在有限的时间 [t0,t1][t_0,t_1][t0​,t1​] 内的测量值唯一确定初始状态 x(t0)x(t_0)x(t0​) ，则称系统在 t0t_0t0​ 时刻可观测。若在任意时刻都可观测，则该系统是状态完全可观测的。

# 对偶原理

# 能控性、能观性与传递函数矩阵的关系

在单输入单输出系统中，系统能控能观的充要条件是传递函数中没有零极点相消的现象。

推论

  1. 一个系统的传递函数所表示的是该系统既能控又能观的那一部分子系统。
  2. 一个系统的传递函数若有零、极点对消现象，则视状态变量的选择不同，系统或是不能控的或是不能观的。
